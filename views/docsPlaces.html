Places

This module provides a set of light weight, easy to use directives for exposing the google places library.

gPlace

The gPlace directive allows for requesting place details for a single place and displaying the information in a custom template. The directive requires 2 things, a template for displaying the information and a variable, defined in your controller, that contains a string with a google place Id.

Use the directive as an element and pass the template url and the id variable as attributes. In your template, access the place details with the details object. For more details on the details object and what information it holds, please see Google Maps API Docs

.controller('PlacesCtrl', function($scope) { // Variable must be a string with a place ID $scope.id = 'ChIJR4dOl_hYwokRApSCaQiBidk'; });

gPlaces

The gPlaces directive works just like the gPlace directive but takes an array of place id's as opposed to just one. The place id's array is defined in your controller and passed to the directive as an attribute just like in the gplace directive.

In your template, use ng-repeat to iterate over the details array. The directive has built in pagination that will activate should you be looking up more than 10 places.
.controller('PlacesCtrl', function($scope) { // Variable must be an array of place id's $scope.idModel = [ 'ChIJR4dOl_hYwokRApSCaQiBidk', 'ChIJv-Ghof5YwokRMtWLEV12hJI', 'ChIJjyX2GqRZwokRT-gdcGoPuSI', 'ChIJqSurReFYwokRec7JFACToas', 'ChIJn8dCo-NYwokRC_4nRUQWbNE', 'ChIJszmN0-JYwokRk-XCDbO6X_Y', 'ChIJt4TrE_1YwokRVedrKxaqYoo', 'ChIJiW0WvwJZwokRIWyzCvo3o5k', 'ChIJsS1xLQJZwokRGfXJPMwXA1A', 'ChIJI5xCX6NZwokR3jdSQwsw2DI', 'ChIJuVE5aLtZwokR-K75OxUEtzI', 'ChIJ7R4tgLtZwokRM8thlhlzE2o', 'ChIJxbWTG_pYwokRPgtFVKi-Cuc', 'ChIJKZVnwFVYwokRgDw_sxw3NCo', 'ChIJw_JUgvhYwokR91EMxVDhB8M', 'ChIJZ3oXOVZYwokRNnAXaDRKAzg', 'ChIJP9idxlZYwokRLH-I1mNfzYQ', 'ChIJndGJ5FNYwokRricJvhT0t1s', 'ChIJhZ0Sn1ZYwokRRA1MZJZHrHA', 'ChIJhTdv51NYwokR7V105uVzf8g', 'ChIJvfVwDFJYwokRtWobbwOMEVM', 'ChIJpY9Tg01YwokRCr_aQpDrqgk', 'ChIJ-fRuLFdYwokR0KKQ6Av_WhQ', 'ChIJh-DIsE1YwokRhuFrdM1ge5E' ]; });
