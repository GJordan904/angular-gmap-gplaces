angular-gmap-gplaces

Modules for integrating the Google maps, places, and directions libraries into an angular application

Angular modules for easy integration of the Google Maps Javascript API, including the Places library and Directions service.

Code on Github

About

angular-gmap-gplaces is a set of modules, written in Javascript, that facilitates easy integration of the Google Maps, Places, and Directions libraries into an angular application.

This package focuses more on the features of the Directions service and the Places library of the Google Maps Javascript API version 3.

I started working on these modules as a way to learn JS, and Angular, and to become more familiar with some Google API's. I am sharing them now in hopes that they may be of use to others and hopefully I can get some feedback on my work

Features

  • Autocomplete
  • Directions
  • Geolocation
  • Map
  • Map Menu
  • Place Details
  • SearchBox
