Code on Github

About

angular-gmap-gplaces is a set of modules, written in Javascript, that facilitates easy integration of the Google Maps, Places, and Directions libraries into an angular application.

I started working on these modules as a way to learn JS, and Angular, and to become more familiar with some Google API's. I am sharing them now in hopes that they may be of use to others and hopefully I can get some feedback on my work
