Getting Started

All you need to get started using the angular-gmap-gplaces modules

Install

First we will need to install the package. This can be done automatically with Bower or manually. If you choose to manually install be sure you have all required dependencies.

Bower

bower install angular-gmap-gplaces

Manual

angular-gmap-gplaces

Includes minified and and non-minified variants. No source files included.

Configure

  1. Add angular-gmap-gplaces as a dependency to your application 
    angular.module('App', ['angular-gmap-gplaces']
  2. Configure the $aggLoader. This provider will load the Google maps API. 
    .config(function($aggLoaderProvider) {
	$aggLoaderProvider.setOptions({
		lang: 'en-US',
		key: 'Your API Key',
		libs: 'places',
		numMaps: 4,
                loadFontAwesome: true
	});
});
  3. Add a resolve for any route using the modules. This ensures the Google Map API is loaded prior to loading any directives requiring it. 
    state('app.home', {
    url: '/home',
    views: {
        'mainContent': {
            templateUrl: 'views/home.html',
            controller: 'HomeCtrl as home'
        }
    },
    resolve: {
        google: '$aggLoader'
    }
})
  4. You are now ready to start using the modules. Head over to the Docs for details on usage.
