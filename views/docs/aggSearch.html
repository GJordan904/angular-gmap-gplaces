agg-search

The agg-search directive is used to turn an input box into a Google Place SearchBox. This SearchBox will present place suggestions based on user input and on selection of a place will update the model with the places details. The returned details may be either a single PlaceResult object or multiple objects returned as a PlaceResult[ ]

For more information see: Autocomplete for Addresses and Search Terms

Requires the Places library

Usage

<input agg-search="obj" type="text" placeholder="Search for places">

Attributes

Attribute Type Details
agg-search Object An object literal containing the model to update and optionally a map to associate with the SearchBox. If a map reference is passed to the SearchBox, results will be biased to the maps viewport.

agg-search

Property Type Details
map google.maps.Map A reference to the map to associate with the SearchBox. If this property is specified then the SearchBox results will be biased to the maps viewport.
model Object The model to be updated with the PlaceResults.
