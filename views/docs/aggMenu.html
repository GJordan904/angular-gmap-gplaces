agg-menu

The agg-menu directive creates a menu button on the map. This button, when clicked, will slide out an agg-search input box that allows searching the map. Results are displayed in a list that slides down from the input box.

In addition to the input box are two buttons, one is an X that clears the map and search results and the other is the directions icon.

When clicked, the directions button will slide the input box out and slide in the directions menu. This menu allows searching for directions between two points by four different travel methods, walking, driving, bicycling, or transit.

The directions menu has two agg-auto-complete input boxes for starting point and destination. Once a directions request has been made and a successful response received, the route will be drawn on the map and the directions will display in a list that slides down from the menu.

For the menu to be positioned properly and for animations to work, the parent agg-map directive must be placed inside of a relatively positioned div with overflow hidden.

Requires the Places Library

Requires the agg-map directive

Usage

<div class="map-container">
    <agg-map options="samples.mMap">
        <agg-menu></agg-menu>
    </agg-map>
</div>
