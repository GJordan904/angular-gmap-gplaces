aggLocationServ

The aggLocationServ service holds methods for watching or getting a users location.

Properties

Property Type Details
gMap google.maps.Map A reference to the map where the location marker is being displayed
marker google.maps.Marker The location marker
watchId int The id returned by the watchPosition call.

Methods

Method Returns Details
getLoc() Promise

Method that checks the users location and returns the coordinates as a promise.

The returned promise is of the form {lat: float, lng: float}
watchLoc(map) void

Method that watches the users location and on successful response draws an AggLocationMarker on the map.

Takes the google.maps.Map where the marker will be displayed as its argument
cancelWatch() void Method for cancelling the current watchPosition call.
