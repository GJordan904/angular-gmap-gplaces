AggLocationMarker

The AggLocationMarker factory extends the google.maps.OverlayView to create a custom location marker.

Constructor

Constructor
AggLocationMarker(options)

Creates an AggLocationMarker and draws it on the map provided in the options at the position specified in the options.

The options parameter takes on object with the following properties:

  • position: LatLng
  • map: google.maps.Map
