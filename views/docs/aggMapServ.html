aggMapServ

The aggMapServ service creates google maps and stores them in an array.

Properties

Property Type Details
maps google.maps.Map[] An array for storing the applications maps. Anytime a map is created it is stored in this array. This allows referencing the map by its index and storing details like zoom, center, and map type so that when a map is destroyed then recreated it will be restored to its previous state.

Methods

Method Type Details
getMap() google.maps.Map Method that creates a Google map with the supplied options. This method first checks to see if the map already exists in the maps array. If the map already exists it will be recreated using its previous zoom, center, styles, and map type.
