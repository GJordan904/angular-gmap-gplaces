aggMapServ

The aggMapServ service creates google maps and stores them in an array.

Properties

Property Type Details
maps google.maps.Map[] An array for storing the applications maps. Anytime a map is created it is stored in this array. The size of array is defined when configuring the module with the $aggLoaderProvider

Methods

Method Type Details
makeMap() google.maps.Map Method that creates a Google map with the supplied options and saves it to the map array.
getMap(int: index) promise Method that checks the maps array for the map at the index provided and returns the map as a promise if it exists. Due to the nature of the directives, this method will check multiple times for the map before returning a rejection. This method is currently not in use and I am not sure it functions completely right at the moment.
