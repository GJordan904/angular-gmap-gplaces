agg-map

The agg-map directive is used to instantiate a Google Map object. The directive accepts an object literal on the options attribute for configuring the map.

Usage

<agg-map options="Object">
    <!-- Directives requiring agg-map go here-->
</agg-map>

Attributes

Attribute Type Details
options Object An object literal containing the options for configuring the map. Below are the additional options unique to the agg-map directive. These are in addition to the standard Google Maps options. The required Google Maps options all have defaults and a default map will display if these options are not set.

Options

Option Type Details
index int The maps index in the array of maps. Used for identifying maps when there are multiple maps in the application. This value defaults to zero and must be set if there are multiple maps in the application.
center LatLng | LatLngLiteral | string The initial map center. Can be set with a LatLng object, a LatLng Literal, or the string 'location'. If set to 'location', the map will be centered on the users location.
mapId string The maps div id. Defaults to map-canvas. If multiple maps are displayed on one page, this value must be unique for each map.
