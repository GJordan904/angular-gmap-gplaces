agg-map

The agg-map directive is used to instantiate a Google Map object. The directive accepts an object literal on the options attribute for configuring the map.

Usage

<agg-map options="{Object}">
    <!-- Directives requiring agg-map go here-->
</agg-map>

Attributes

Attribute Type Details
options Object An object literal containing the options for configuring the map. Below are the additional options unique to the agg-map directive. These are in addition to the standard Google Maps options. The required Google Maps options all have defaults and a default map will display if these options are not set.

Options

Option Type Details
index int The maps index in the array of maps. Used for identifying maps when there are multiple maps in the application. This value is set automatically starting at 0 for the first map and auto incrementing with each additional map
mapId string The maps div id. Defaults to map-canvas
