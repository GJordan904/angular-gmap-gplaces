agg-auto-complete

The agg-auto-complete directive is used to turn an input box into a Google Place Autocomplete Box. This Autocomplete box will present place suggestions based on user input and on selection of a place will update the model with the places details.

For more information on the place details results see: Autocomplete for Addresses and Search Terms

Requires the Places library

Usage

<input agg-auto-complete="model.to.update" type="text" placeholder="Search for a place">

Attributes

Attribute Type Details
agg-auto-complete Object The model to update with the returned PlaceResult.
