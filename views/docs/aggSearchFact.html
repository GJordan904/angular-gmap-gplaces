aggSearchFact

The aggSearchFact factory provides methods for performing place searches and for handling the search results

Properties

Property Type Details
searchObj.markers google.maps.Marker[] An array for storing markers generated by the mapSearch method.
searchObj.results google.maps.places.PlaceResult[] An array for storing PlaceResult objects returned by searches.

Methods

Method Returns Details
nearbySearch(options) Promise

Method that performs a search using the details provided as the options argument. Returns a PlaceResult[ ] as a promise.

The options parameter takes an object with the following properties.

  • location: LatLng|LatLngLiteral
  • radius: int
  • type: string
  • minPriceLevel: int

For more information see: Nearby Search Requests.
mapSearch(places, map) Promise Method that takes a PlaceResult[ ] and a Google Map and creates markers and Info Boxes for each place.
