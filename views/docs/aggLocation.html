agg-location

The agg-location directive uses the Geolocation API to retrieve the users location and adds a custom marker to the map at that location. The marker is a custom OverlayView created by the AggLocationMarker factory.

This directive requires the users permission for access to their location. In Chrome, the Geolocation API will only work on sites served from a secure context. (ie. HTTPS)

Requires the agg-map directive

Usage

<agg-map options="Object">
    <agg-location></agg-location>
</agg-map>
